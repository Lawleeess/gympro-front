¿Qué ofrecemos?

En nuestro gimnasio lo más importante eres TÚ así que además de ofrecerte un equipo de primera calidad te ofrecemos algunas características para mejorar tu rendimiento en el GYM, algunas de estas características son:

  • Pago desde la Página Web

    Podras realizar tu pago mensual desde la página web utilizando PayPal

  • Seguridad

    Podras estar seguro de tu información gracias a nuestro sistema de verificación de correo

  • Página intuitiva

    Nuestra página cuenta con componentes muy intuitivos y fácil de usar

  • Módulo de información

    Nuestra página cuenta con un apartado donde podrá subir su información

  • Módulo de rutinas

    En nuestra página web podrás consultar, agregar y eliminar rutinas en cualquier momento

  • Módulo de Seguimiento

    En este módulo tendrás resultados del progreso que llevas cada mes
